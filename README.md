# Boilerworks Go + Next.js

> API-first Go backend with Next.js 16 frontend for performance-sensitive applications.

**Status:** Planned

Go with Chi or Echo as an API backend paired with Next.js 16 for teams that want Go's raw performance, small memory footprint, and straightforward concurrency model behind a modern React frontend. Choose this when latency and resource efficiency matter more than framework features.
